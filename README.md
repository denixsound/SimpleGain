# SimpleGain

Простий та ефективний аудіоплагін для зміни гучності (Gain), створений на базі фреймворку **JUCE (C++)**. Цей проект є повністю відкритим (Open-Source) та розповсюджується під ліцензією GPLv3.

## 🚀 Особливості
* **Чистий DSP код:** Проста та оптимізована обробка аудіосигналу без затримок.
* **Кросплатформеність:** Завдяки JUCE, плагін може бути скомпільований під macOS та Windows.
* **Формати:** Підтримка VST3 та AU (Audio Unit) для використання у будь-якій сучасній DAW (Ableton Live, Logic Pro, FL Studio, Reaper тощо).

## 🛠 Інструменти розробки
* **Мова:** C++
* **Фреймворк:** JUCE 8 (Starter / Open-source ліцензія)
* **IDE:** Xcode (для macOS) / Visual Studio (для Windows)

## 📂 Структура репозиторію
* `/Source` — вихідний код плагіна (`.cpp` та `.h` файли процесора та інтерфейсу).
* `/JuceLibraryCode` — конфігураційні файли, згенеровані Projucer.
* `SimpleGain.jucer` — головний файл проекту для утиліти Projucer.

## 🔨 Як скомпілювати самостійно
1. Завантажте та встановіть [JUCE](https://juce.com/).
2. Відкрийте файл `SimpleGain.jucer` через програму **Projucer**.
3. Переконайтеся, що обрано правильний експортер (Xcode для Mac або Visual Studio для Windows).
4. Натисніть кнопку **Save and Open in IDE** для генерації та відкриття проекту у вашому середовищі розробки.
5. Скомпілюйте проект у режимі `Release`. Готовий файл плагіна з'явиться у вашій системній папці плагінів.
